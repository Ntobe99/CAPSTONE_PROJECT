routes description!

Method Path Description
GET / To render the home page (index.html)
GET /users To display all users.
GET /user/:id To display a single user.
POST /add Used to send user data to a server, by adding a new record.
PUT /user/:id Used to update your user data.
DELETE /user/:id Used to delete user data from the server.
GET /products Used to display all products.
GET /product/:id Used to display a single product.
POST /product Used to send product data to the server, by adding a new record.
PUT /product/:id Used to update your products data.
DELETE /product/:id Used to delete product data from the server.
GET /user/:id/cart see all cart items
POST /user/:id/cart Add item to cart
PUT /user/:id/cart update cart item
DELETE /user/:id/cart delete cart item
